# Poker Hand Map

6-max 캐시 게임을 위한 프리플랍 핸드 레인지 학습 도구입니다.

## Features

- **Hand Map**: 포지션별 오픈 레인지, 3-bet 레인지를 시각적으로 확인
- **Quiz**: 핸드 비교, 오픈 레인지, 카테고리 분류 등 다양한 퀴즈 모드
- **Speed Quiz**: 시간 제한 퀴즈로 빠른 판단력 훈련
- **Spaced Repetition**: Leitner 박스 기반 간격 반복 학습
- **Dashboard**: 학습 진행도 및 통계 확인

## Tech Stack

- React 19
- TypeScript
- Vite
- Tailwind CSS 4
- React Router

## Getting Started

```bash
npm install
npm run dev
```
